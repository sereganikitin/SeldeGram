Сюда нужно положить два mp3-файла:

- `ringtone.mp3` — звук входящего звонка (луп, ~30 сек)
- `dialing.mp3` — гудок при исходящем звонке (луп, ~5 сек)

Бесплатные источники:
- https://pixabay.com/sound-effects/search/ringtone/
- https://freesound.org/

Если файлов нет — звонки всё равно работают, просто без звукового сопровождения.
